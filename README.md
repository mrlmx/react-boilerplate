# react-boilerplate

A react boilerplate with mobx, typescript and webpack.

## 概述

- [x] VS Code：使用 VS Code 进行开发。
- [x] Yarn：使用 Yarn 管理包依赖。
- [x] Prettier：使用 Prettier 格式化代码。
